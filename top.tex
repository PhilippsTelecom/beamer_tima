\section{Main features}

\subsection{About this template}

\frame{
  I designed this template for my PhD defense with the following
  features :
  
  \begin{itemize}
  \item Color theme matching the TIMA logo colors. Integration of TIMA
    logo.
  \item The presentation structure defines the frame titles.
  \item Progression in the presentation explicited at global level
    (slide numbering), section level (section names) and subsection
    level (progress bar per section).
  \item Fast navigation through slides.
  \end{itemize}

\subsection{Known caveats}

\frame{
  \begin{itemize}
  \item The TIMA logo is a modified version. The font has been altered
    during vectorization. The logo has been cropped on the left to get
    the right blue/turquoise ratio.
  \item Spacing in the section bar is static. The width of the bar can
    be changed at \url{template/beamerouterthemesmalltree.sty} (line
    131).
  \item Slide numbering may not correctly work with appendices.
  \item Dirty relative import of custom color and outer themes.
  \end{itemize}
}
